--------------------------------------------------------------------------
            RF3 Command Line Compiler for RenderWare Graphics
--------------------------------------------------------------------------

This tool takes one or more .rf3 files and converts these to 
RenderWare Assets such as .rws, .dff, .anm, .bsp etc. using 
the given Project/Asset Template files.


The RF3 Command Line Compiler document, rf3cc.pdf, can be found in 
docs\tools in the RenderWare Graphics directory.
