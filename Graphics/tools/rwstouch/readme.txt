--------------------------------------------------------------------------
           File Version Update Tool for RenderWare Graphics
--------------------------------------------------------------------------

rwstool
rwstouch updates RenderWare files to the current version.

Supported chunk IDs:
    rwID_CLUMP
    rwID_ATOMIC
    rwID_WORLD
    rwID_HANIMANIMATION
    rwID_DMORPHANIMATION
    rwID_MTEFFECTDICT
    rwID_PITEXDICTIONARY

Usage: rwstouch [options] [files] ...
   -h, --help       : Prints usage message
   -v, --verbose    : Prints out log messages while processing
