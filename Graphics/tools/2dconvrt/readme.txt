--------------------------------------------------------------------------
         Maestro's 2d Converter Tool for RenderWare Graphics
--------------------------------------------------------------------------

2dconvrt is primarily an importer of Flash .swf files into the 
RenderWare Graphics .anm format. These files may then be played back with 
the 2dviewer viewer, which uses the Rt2dAnim library.

The 2d Converter Tool document, 2dconverter.pdf, can be found in 
docs\tools in the RenderWare Graphics directory.
