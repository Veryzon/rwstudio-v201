About this tool

The PurgeDeleted tool lists the XML files in a RenderWare Studio game database that contain a <Deleted/> element, indicating that the corresponding object has been deleted.

We recommend that you run this tool periodically to clean up these files.

If you are using NXN alienbrain: This tool only deletes files on your PC. After running this tool, remember to remove the copies of the files on the server.

You can run this tool via either:

  • PurgeDeleted.wsf (a Windows script file)

    or

  • PurgeDeleted.hta (an HTML application)

Windows script command-line options

The Windows script file offers several command-line options. To see these options, open a command prompt window, change to the directory containing the script, and then enter:

PurgeDeleted.wsf /?

For example, if you run the Windows script file with the /project:myproject.rwstudio option, then the tool immediately scans for deleted objects in that game database, rather than prompting you to select a project. This is especially useful if you typically work on the same project all of the time. You might even want to create a shortcut that invokes this tool for your project. Here is the Target property from such a shortcut (shown here split over several lines, for clarity):

C:\WINDOWS\SYSTEM32\WSCRIPT.EXE
"C:\RW\Studio\Programs\Tools\PurgeDeleted\PurgeDeleted.wsf"
/project:"C:\Studio Projects\ProjectX.rwstudio"

Running in batch mode

To run this tool without displaying a window, append a //B switch to the Windows script file command line.
